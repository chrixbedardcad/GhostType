How It Works

1 Select text in any application (or place your cursor in a text field to process all text).
2 Press the action hotkey (default: Ctrl+G on Windows/Linux, Cmd+G on macOS).
3 GhostSpell captures the text, sends it to the active AI model with the selected prompt, and replaces it with the result.
4 Press Ctrl+Z to undo and restore the original text.

Status Indicators

GhostSpell appends emoji indicators in your text to let you know what happened:

👻✅No changes needed — the AI returned the same text. Your text is already good!
👻❌Error — the AI provider returned an error (bad API key, network issue, etc.).
👻⏳Timeout — the request took too long. Try again or check your connection.
👻⚠️Text too long — the selected text exceeds the max input character limit. Reduce your selection or increase the limit in General settings.
👻🫥Empty text — no text was captured. Make sure text is selected or the cursor is in a text field.

All indicators can be removed with Ctrl+Z (undo).

Sound Feedback

🔊Working loop — plays while the AI is processing your text.
🎵Success — the text was processed and replaced successfully.
🔇Error — something went wrong (see indicators above).

Toggle sounds on/off in the General tab.

Key Concepts

Provider The AI service that processes your text. Can be a cloud service (OpenAI, Anthropic, Gemini, xAI, DeepSeek) that requires an API key or login, or a local engine (GhostAI, Ollama, LM Studio) that runs on your machine for full privacy. Set up providers in the Providers tab.
Model The specific AI brain within a provider. Each provider offers different models with trade-offs in speed, quality, and cost. For example, OpenAI has GPT-5-mini (fast) and GPT-5.4 (best quality). GhostAI has Qwen3.5-2B (light) and Qwen3.5-4B (more capable). Configure models in the Models tab.
Prompt The instruction that tells the AI what to do with your text. "Correct" fixes spelling, "Polish" improves clarity, "Ask" answers questions. You can create custom prompts for any task. Each prompt can optionally use a different model. Manage prompts in the Prompts tab.

Default Prompts

✏️Correct — fixes spelling and grammar only, preserves tone and style.
💎Polish — improves clarity, structure, and flow while keeping the meaning.
😄Funny — rewrites the text to be witty and entertaining.
📝Elaborate — expands terse text with more detail and context.
✂️Shorten — condenses the text, removes filler and redundancy.
🌐Translate — translates the text to English.
❓Ask — answers questions clearly and concisely.

Add, edit, reorder, or delete prompts in the Prompts tab. Each prompt can use a different model.

Hotkeys

⌨️Action (default: Ctrl+G on Windows/Linux, Cmd+G on macOS) — process selected text with the active prompt.
🔄Cycle Prompt (default: Ctrl+Shift+T / Cmd+Shift+T) — switch to the next prompt. Shows a brief indicator pill.
🛑Cancel — press the action hotkey again while processing to cancel and restore text.

Configure hotkeys in the Hotkeys tab. Right-click the tray icon to see and switch the active prompt.

Tips

• Select text first — GhostSpell processes only your selection. If nothing is selected, it captures all text in the window.

• Works in any application — browsers, editors, chat apps, email.

• The text is always processed in its original language (except the Translate prompt).

• Clipboard contents are preserved and restored after processing.

• You can set up multiple AI providers (OpenAI, ChatGPT Login, Anthropic, Gemini, xAI, DeepSeek, Ollama, GhostAI) in the Providers tab.

• For local/private processing, use GhostAI (built-in) — your text never leaves your machine.
